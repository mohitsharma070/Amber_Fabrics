# Logo Quick Reference - Amber Fabrics

## 🎯 Which Logo Should I Use?

### For Website Header (Dark Background) ✅
**USE:** `logo.svg` (Icon only)  
**Why:** Clean, professional, pairs perfectly with brand name text  
**Size:** 52px height

---

### For Documents
**USE:** `logo-brand.svg` (Full logo - dark version)  
**Why:** Complete branding, professional for business documents  
**Where:** Letterheads, quotations, export documents

---

### For Dark Backgrounds (Presentations, Social Media)
**USE:** `logo-brand-light.svg` (Full logo - light version)  
**Why:** High contrast, fully visible on dark backgrounds  
**Where:** Presentation slides, social media posts, email signatures

---

### For Small Spaces (Favicon, Profile Pictures)
**USE:** `logo-icon.svg` (Icon only)  
**Why:** Recognizable at small sizes  
**Where:** Browser favicon, app icons, social media profile pictures

---

### For Product Packaging
**USE:** `logo-brand.svg` (Full logo - dark version)  
**Why:** Complete brand identity on product labels  
**Where:** Fabric labels, hang tags, packaging boxes

---

## 📊 Logo File Summary

| File | Type | Background | Use Case |

## ⚡ Quick Actions

### Change Website Logo
1. Go to: **Admin → Site Settings**
2. Upload logo file
3. Click "Save Settings"

### Download All Logos
All logo files are in: `images/` folder
- logo.svg
- logo-icon.svg
- logo-brand.svg
- logo-brand-light.svg

## 📐 Sizing Guide

### Digital
- **Website:** 48-56px height
- **Email:** 200-300px width
- **Social Media:** 400-600px width

### Print
- **Business Card:** 40-50mm
- **Letterhead:** 60-80mm
- **Packaging:** 30-60mm
