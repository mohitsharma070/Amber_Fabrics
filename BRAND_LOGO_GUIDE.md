## 🎯 Design Elements Explained

### **Icon: Handloom Weave**
- **Frame:** Represents traditional handloom structure
- **Vertical lines (Warp threads):** 6 parallel threads representing fabric foundation
- **Horizontal waves (Weft threads):** Interwoven pattern showing craftsmanship
- **Gold corners:** Premium quality indicators
- **Shuttle (ellipse):** Traditional weaving tool in the center

### **Typography**
- **"Amber Fabrics":** Georgia serif font - elegant, traditional, trustworthy
- **"TEXTILES":** Segoe UI sans-serif - modern, professional, clean
- **Tagline:** "SUSTAINABLE CRAFTSMANSHIP" - brand values

### **Color Palette**

#### Primary Colors:
- **Deep Teal Green:** `#0f766e` - Eco-friendly, natural, textile heritage
- **Teal:** `#14b8a6` - Modern, fresh, sustainable

#### Accent Colors:
- **Gold:** `#d4af37` - Premium quality, craftsmanship, luxury
- **Light Gold:** `#fbbf24` - Warmth, authenticity

#### Supporting Colors:
- **White:** `#ffffff` - Purity, cotton, cleanliness
- **Dark Teal:** `#115e59` - Depth, reliability
- **Gray:** `#6b7280` - Professionalism, balance

## 💼 Usage Guidelines

### Website Header
**Current Setup:** Icon-only logo (`logo.svg`) + Brand name text  
**Recommendation:** Keep current setup - clean and professional

**Alternative:** Use full `logo-brand-light.svg` and remove text duplication

### Business Documents
- **Invoices:** Use `logo-brand.svg` (top left)
- **Quotations:** Use `logo-brand.svg` (letterhead)
- **Export Documents:** Use `logo-brand.svg`

### Digital Marketing
- **Social Media Posts:** Use `logo-brand-light.svg` or `logo-brand.svg` depending on background
- **Email Campaigns:** Use `logo-brand.svg` (600px width recommended)
- **Profile Pictures:** Use `logo-icon.svg`

### Product Branding
- **Fabric Labels:** Use `logo-brand.svg` or icon-only for small labels
- **Packaging:** Use full logo with tagline
- **Hang Tags:** Use `logo-brand.svg`

### Print Materials
- **Business Cards:** Use full logo (front) or icon (back)
- **Brochures:** Use full logo on cover
- **Catalogs:** Use full logo on title page

## 📐 Size Recommendations

### Digital
- **Website Header:** Icon 48-60px height
- **Email Signature:** Full logo 200-250px width
- **Social Media Cover:** Full logo 400-600px width
- **Profile Picture:** Icon 400x400px minimum

### Print
- **Business Card:** 40-50mm width
- **Letterhead:** 50-70mm width
- **Envelope:** 30-40mm width
- **Fabric Label:** Minimum 20mm width

---

## 🚫 Don'ts - Protect the Brand

❌ Do not stretch or distort the logo  
❌ Do not change the color scheme without approval  
❌ Do not add drop shadows or effects  
❌ Do not rotate the logo  
❌ Do not place on busy backgrounds without proper contrast  
❌ Do not separate the icon from the text in full logo versions  
❌ Do not recreate or modify the handloom pattern  

---

## ✅ Do's - Best Practices

✅ Maintain minimum clear space around logo (equal to icon height)  
✅ Use SVG format for web and digital use  
✅ Convert to high-res PNG (300 DPI) for print  
✅ Ensure good contrast with background  
✅ Use appropriate version for background color  
✅ Scale proportionally  
✅ Test readability at actual size before production

## 🔄 How to Switch Logo Versions

### Via Admin Panel (Easiest):
1. Navigate to **Admin Dashboard → Site Settings**
2. Upload desired logo file (logo-brand-light.svg or logo-icon.svg)
3. Click "Save Settings"

### Via File System:
1. Copy desired logo file
2. Rename to `logo.svg`
3. Replace in `images/` directory

### Current Active Logo:
`logo.svg` = Professional handloom icon (86x100)
